A simple and lightweight plugin to add a <strong> User Dashboard Page</strong> to the easy digital downloads plugin.

== Description ==

Easily create a user dashboard for your easy digital downloads store by adding [edd-dash] shortcode to a page.
Also, this plugin has a Gutenberg block to add the shortcode to a page or post to create a customer dashboard.

Tabs included:

* Profile Edit
* Download History
* Purchase History
* Access Pass details
* Subscriptions
* Licenses

Some of the tabs are just visible when a certain plugin is active.


== Installation ==

You can install User Dashboard-Easy Digital Downloads by either using wordpress.org plugin directory or by directly uploading plugin files to your server.

== Frequently Asked Questions ==

= How can I use this plugin?

It's really easy just to add this shortcode [edd-dash] to a page of your website and make sure the easy digital downloads plugin is active.
